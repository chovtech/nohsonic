# Gemini Image Generation Prompts — NOHSONIC Service Pages

## How to use

1. Open Google AI Studio (aistudio.google.com) → select **Imagen 3**
2. In the settings panel, set the **aspect ratio** to match each image type before generating
3. Copy the prompt, paste it, generate
4. Save files using the naming convention and folder structure shown
5. Place all files inside: `app/website/images/services/{service-slug}/`

## Important note on logo rendering

AI image generators cannot reliably render specific text or logos on uniforms and signage — the spelling will often be wrong or distorted. The prompts below tell Gemini to show branded uniforms and clinic signage, but you will likely need to **overlay the actual NOHSONIC logo in post** using Canva or Photoshop on any image where the logo matters. Generate the image first, then add the logo on top.
